I create generative art under the identity Ten Twentythree (after my birthday, October 23). I discovered it in the summer of 2022.

* What is generative art?

In my words, generative artwork is an algorithm that creates some space of artistic output. As a generative artist, my role is to write code that controls this space while still producing diverse and interesting works.

* Is this AI art (ChatGPT/Midjourney/Sora/StableDiffusion -generated)?

No, it's all hand-coded using p5.js, a JavaScript library for creative coding.
